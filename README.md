# Currency Converter

A simple Currency Converter built using HTML, CSS, and JavaScript.

## Features

- Convert between different currencies
- Live exchange rates using an API
- Country flags for selected currencies
- Swap currencies
- Input validation
- Simple and responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Fetch API

## Live Demo

https://jamhambo.github.io/Currency-Converter-MP/
## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/JAMHAMBO/Currency-Converter-MP.git
   ```

2. Open `index.html` in your browser.
